## Introduction
This repository holds the basic implementation of a custom ERC20 Token. The project was implemented with [Truffel](https://www.trufflesuite.com/).

**Execute tests**
```
npx truffle test
```
**Deploy**
```
truffle migrate 
```
**Update**
```
truffle migrate --reset
```
